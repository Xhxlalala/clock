# clock

本项目主要是为了学习 Cursor 的使用，所有代码均有 Cursor 完成。  

项目使用 HTML + CSS + JS 实现了一个钟表界面，主要有以下特点：

- 每次打开都会显示当前时间
- 输入框支持手动输入时间进行设定
- 每过一分钟表盘颜色会变化一次
